JS Basics

Erste JS Kommmandos

  • Ein einzelnes Kommando endet immer mit einem 
    
        alert('Hallo');
  • Wörter (Strings oder Zeichenketten) werden mit 
    
        "Hallo!"
        'Dominik'
    umschlossen
  • Variablen werden so definiert 
    
        const eineFrage = "was los?";
        const eineZahl = 47.11;
        let einWort = 'Wieso';
        let eineAndereZahl = 2031;
  • Ausgaben und testen von Ausgaben mit 
    
          console.log('Dieser Text wird ausgegeben')
  • Nummern werden einfach so mmit einem Punkt notiert 
    
        const eineKommazahl = 12.3
        const eineGanzzahl = 99
  • Wir können Unterscheidungen treffen 
    
          const einWort = prompt('Gib ein Wort ein');
          if (einWort === 'hallo') {
            alert('Du hast hallo eingegeben')
          } else {
            alert("Du hast etwas anderes eingegeben")
          }
  • Wir können auch Loops (deutsch Schleifen) bauen 
    
          let zahl = 0;
          while (zahl < 10) {
            zahl = zahl + 1;
            alert('Durchlauf nr' + zahl);
          }
